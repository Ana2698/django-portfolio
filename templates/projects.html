Preview of Projects

Fake News Detector

Performed the task of classifying news as fake or real news using the ISOT(Information Security and Object Technology Lab) Fake news dataset found on Kaggle. NLP techniques(Tokenization, Vectorization, Word embedding etc.) used for text preprocessing and feature extraction. Classifier models (Linear SVC, Linear Regression and MultinomialNB) and Neural Network models(Basic LSTM, LSTM with GLoVe embeddings and Bidirectional LSTM) were trained, tested and validated.Validation techniques such as K-Fold validation and custom dataset used to validate the models. Performance of all the models were compared based on the testing and validtaion results. The results can be viewed on GitHub.

COVID-19 Infection Predictor

This projects is the implementation of mainly two tasks. One task is to predict whether an individual is infected with Covid-19 virus and the other is to predict the severity of the infection based on given symptoms. To accomplish both the tasks a Decision Tree classifier model is trained and its parameter values are decided by applying hyperparameter tuning. The dataset used to train the models for the two tasks is generated based on the WHO guidelines and Ministry of Health and Family Welfare, India and can be found on Kaggle

Handwritten Number Recognition

Implemented the task of classifying of handwritten digits. Several classifier (K-Nearest Neighbor, Random Forest, Support Vector Machine) and neural networks(Convolution Neural Network) models were trained on the MNIST dataset which is a widley used dataset of handwritten digits. The model performances are test and validated using K-Fold validation.

Data Engineering and Machine learning on high-volume gene data

Built clustering model for high volume RNA-sequence data. Performed data pre-processing by using techniques like dimensionality reduction through tools like PCA and FastICA. Trained a K-Means model with the number of clusters as 16, to get achieve an accuracy of 0.86
